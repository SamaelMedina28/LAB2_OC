Pasos para ensamblar con NASM:

Comando para ensamblar: nasm -f elf <archivo.asm>
Comando para enlazar: ld -m elf_i386 <archivos.o> -o <output_file>
Ejemplo
ld -m elf_i386 p7.o ../LIB/libpc_io.a -o p7
